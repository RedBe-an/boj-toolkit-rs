# boj-toolkit-rs

백준 온라인 저지(Baekjoon Online Judge)에서 문제를 해결하기 위한 Rust 명령줄 툴킷입니다.

## 소개

boj-toolkit-rs는 백준 온라인 저지 플랫폼에서 문제를 더 효율적으로 해결할 수 있도록 설계된 강력한 Rust 기반 명령줄 도구입니다. 이 도구킷은 문제 정보 검색, 코드 템플릿 생성, 제출 관리 등 다양한 기능을 제공하여 프로그래밍 경험을 향상시킵니다.

## 주요 기능

- **문제 정보 조회**: 문제 ID를 사용하여 상세 정보를 빠르게 검색합니다.
- **코드 템플릿 생성**: 선택한 프로그래밍 언어에 맞는 기본 코드 구조를 자동으로 생성합니다.
- **제출 관리**: 코드 제출 및 결과 확인을 간소화합니다.
- **성능 분석**: 제출한 코드의 실행 시간과 메모리 사용량을 분석합니다.
